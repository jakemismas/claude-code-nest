# Folders and tags

A chat lives in exactly one folder (its home) and can carry any number of tags.

- New Folder accepts a slash name like Work/ClientA to create a folder and one
  subfolder (the panel shows and creates one sublevel).
- New Tag creates a reusable label; right-click a chat for its tag menu and color
  picker, or right-click a folder to recolor or rename it.
- Drag a chat onto a folder to move its home, onto a tag chip to add the tag, or
  onto Unsorted to clear its folder.
- Link a chat under another with Link to Chat... to record a parent-child
  relationship (a future panel surface will render the nesting).

All of this is your own organization layer. It is stored separately from your
transcripts and synced best-effort across machines.
