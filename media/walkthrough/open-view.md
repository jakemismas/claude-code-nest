# The Nest panel

Claude Code Nest adds its own Activity Bar panel for the active workspace. It is a
single view, Organize, and it is the whole surface:

- A Starred section, a Questions section (chats with an unread assistant turn that
  asks something, a scan-time heuristic labeled as such), your folder hierarchy with
  per-folder color and counts, and an always-present Unsorted bucket.
- Tag filter chips, full-text search, sort, folder rename, and drag-and-drop.
- Read markers on each row: a blinking question badge when Claude's last message asks
  you something, and a solid dot when Claude replied and you have not looked yet.
- Two in-panel overlays: Settings (the gear in the toolbar) sets your keep window and
  which sections show, and Archive (the bottom "Archived" row) lists chats you
  archived, each kept as a Nest-owned copy that survives Claude Code's own transcript
  cleanup.

Everything is read-only over your Claude Code transcripts. Nest never writes,
renames, moves, or deletes anything under your sessions.
