# The Nest panel

Claude Code Nest adds its own Activity Bar panel for the active workspace:

- Organize: the browsing surface, with Starred and Questions sections, your
  folder hierarchy, tag filter chips, search, sort, and drag-and-drop.
- Archive: chats you archived, each kept as a Nest-owned copy that survives
  Claude Code's own transcript cleanup.

Everything is read-only over your Claude Code transcripts. Nest never writes,
renames, moves, or deletes anything under your sessions.
